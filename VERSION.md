## 🚀 使用说明

1. **启动服务器**
   ```bash
   cd /Users/yangwang/webgames/gba-emulator
   python3 -m http.server 8080
   ```

2. **打开浏览器**
   - 访问：`http://localhost:8080`

3. **加载游戏**
   - 点击"📂 选择游戏"按钮
   - 或从侧边栏选择 ROM 文件
