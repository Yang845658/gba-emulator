# 🎮 GBA 模拟器 - 使用说明

## ⚠️ 重要：必须使用 HTTP 服务器

**不能直接用浏览器打开 HTML 文件！** 由于浏览器安全限制（CORS），必须使用 HTTP 服务器访问。

## 🚀 启动步骤

### 1. 打开终端

```bash
cd /Users/yangwang/webgames/gba-emulator
```

### 2. 启动 HTTP 服务器

**Python 3（推荐）：**
```bash
python3 -m http.server 8080
```

**Python 2：**
```bash
python -m SimpleHTTPServer 8080
```

**Node.js（如果安装了 http-server）：**
```bash
npx http-server -p 8080
```

### 3. 打开浏览器

访问：**http://localhost:8080**

### 4. 加载游戏

**方法 1 - 本地文件（推荐）：**
1. 点击底部"📂 选择游戏"按钮
2. 选择您的 .gba 文件
3. 游戏立即开始

**方法 2 - ROM 列表（仅 HTTP 环境）：**
1. 点击右上角"⚙️"设置图标
2. 打开"游戏列表"侧边栏
3. 点击 ROM 文件加载

## 🎮 控制方式

### 键盘控制

| GBA 按键 | 键盘 | 说明 |
|---------|------|------|
| A | Z | 主要动作 |
| B | X | 次要动作 |
| SELECT | Shift | 选择 |
| START | Enter | 开始 |
| ↑↓←→ | ↑↓←→ | 方向键 |
| L | Q | 左肩键 |
| R | W | 右肩键 |

### 触摸控制

- **方向键** - 左侧十字键
- **A 键** - 红色圆形按钮
- **B 键** - 蓝色圆形按钮
- **SELECT/START** - 右下方小按钮
- **L/R 键** - 顶部肩部按钮

## ⚙️ 设置

点击右上角"⚙️"打开设置：

- **屏幕方向** - 自动/横屏/竖屏
- **音效** - 开关游戏声音
- **音量** - 0-100% 调节
- **按键音** - 开关按键提示音
- **显示性能** - 显示实时 FPS

## 💾 保存/读取

使用底部按钮：
- **保存** - 保存当前进度
- **读取** - 读取保存的进度

> 保存数据存储在浏览器中，清除浏览器数据会丢失存档。

## 📁 添加游戏

将 .gba 文件放入 `roms/` 文件夹：

```
gba-emulator/
└── roms/
    ├── 最终幻想-中文版.gba
    ├── EUR-KIRH.GBA
    └── 您的游戏.gba
```

刷新页面后，游戏会出现在侧边栏列表中。

## ❓ 常见问题

### Q: 显示"CORS"错误？
**A:** 您一定是直接用 `file://` 打开的。请使用 HTTP 服务器（见上方步骤）。

### Q: 游戏加载后黑屏？
**A:** 
1. 等待几秒钟，游戏需要加载
2. 检查 ROM 文件是否完整
3. 尝试刷新页面

### Q: 触摸没有反应？
**A:** 
1. 确保游戏已加载
2. 检查浏览器是否支持触摸
3. 尝试刷新页面

### Q: 声音很小或没有？
**A:** 
1. 检查设置中的"音效"是否开启
2. 调节"音量"滑块
3. 检查设备音量

## 📱 移动端使用

1. 确保手机和电脑在同一网络
2. 在电脑上启动 HTTP 服务器
3. 在手机浏览器访问：`http://电脑IP:8080`
4. 建议使用横屏模式

## 🔧 技术细节

- **模拟器核心** - EmulatorJS (mGBA)
- **渲染** - HTML5 Canvas / WebGL
- **音频** - Web Audio API
- **存储** - localStorage / IndexedDB

## ⚠️ 注意事项

1. **首次加载** - 需要下载模拟器核心（约 5-10MB）
2. **网络要求** - 需要访问 cdn.emulatorjs.org
3. **浏览器要求** - Chrome/Safari/Firefox 最新版
4. **ROM 合法性** - 请确保您拥有游戏的合法使用权
