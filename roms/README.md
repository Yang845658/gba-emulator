# 📂 ROM 文件夹

请将您的 GBA 游戏 ROM 文件（.gba 格式）放入此文件夹中。

## 如何添加游戏

1. 获取 `.gba` 格式的游戏文件
2. 将文件复制到此 `roms/` 文件夹
3. 刷新浏览器页面
4. 点击设置图标 ⚙️ 打开侧边栏，即可看到游戏列表

## 支持的文件格式

- ✅ `.gba` - Game Boy Advance ROM

## 文件命名建议

- 使用英文或拼音命名，避免乱码
- 例如：`pokemon-emerald.gba`、`mario-kart.gba`
